# サンプルプロトコル（テスト用）

動作確認・デモ用のコピペ可能なプロトコルとシード論文 PMID。すべて実在する腹臥位療法 ARDS の RCT で、捕捉率（capture rate）の挙動確認に使える。

## Research Question（RQ）

成人の中等症〜重症 ARDS 患者において、腹臥位療法は仰臥位管理と比べて死亡率を改善するか？

## フレームワーク種別

PICO

| 項目 | 内容 |
|---|---|
| **P (Population)** | 成人（18歳以上）の急性呼吸窮迫症候群（ARDS）患者。中等症〜重症（PaO2/FiO2 ≤ 200 mmHg） |
| **I (Intervention)** | 腹臥位療法（prone positioning / prone ventilation） |
| **C (Comparison)** | 仰臥位での通常管理（supine positioning） |
| **O (Outcome)** | 28日・90日死亡率、ICU滞在日数、人工呼吸器離脱日数 |

## 組入基準（Inclusion criteria）

- 成人 ICU 患者
- ARDS 診断（Berlin 定義または AECC 定義）
- ランダム化比較試験（RCT）

## 除外基準（Exclusion criteria）

- 小児・新生児
- 妊婦
- 腹臥位の禁忌（不安定な脊椎損傷、頭蓋内圧亢進等）
- 観察研究・症例報告
